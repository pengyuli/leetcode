# First Bad Version
## 思路 (二分法)
这道题要寻找满足条件的target中的第一个, 所以用模板1可能会错过target, 所以选用第二个模板
## 复杂度
time:O(log(n))
## 代码
```java
public class Solution extends VersionControl {
    public int firstBadVersion(int n) {
        int left = 1, right = n;
        while (left + 1< right) {
            int mid = left + (right -left) /2;
            if (isBadVersion(mid)) {
                right = mid;
            } else {
                left = mid;
            }
        }
        if (isBadVersion(left)) {
            return left;
        } else {
            return right;
        }
    }
}

```
