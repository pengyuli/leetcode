3Sum Closest

3Sum Closest

思路 双指针

和3Sum相同,还是一个指针遍历数组, 另外两个用双指针向中间夹逼. 因为只需要一个最符合的答案所以可以忽略所有重复的情况.

复杂度

时间 O(n^2) 空间 O(1)

代码

class Solution {
    public int threeSumClosest(int[] nums, int target) {
        int res = nums[0] + nums[1] + nums[2];
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 2; i++) {
            int left = i + 1, right =  nums.length -1;
            while (left < right) {
                int sum = nums[i] +nums[left] + nums[right];
                if (sum > target) {
                    right --;
                } else if (sum < target) {
                    left++;
                } else {
                    return target;
                }

                if (Math.abs(target - res) > Math.abs(target - sum)) {
                    res = sum;
                }

            }
        }
        return res;
    }
}
