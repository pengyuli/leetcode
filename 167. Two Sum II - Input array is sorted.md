# Two Sum II - Input array is sorted
## 方法 TwoPointer
因为数组是有序的, 用双指针的方法从左右向中间找到结果
## 复杂度
时间O(n) 空间O(1)
##代码
``` java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] res = {-1,-1};
        int left = 0, right = nums.length - 1;
        while (left < right) {
            int sum = nums[left] +nums[right];
            if (sum > target) {
                right--;
            } else if (sum < target) {
                left++;
            } else {
                res[0] = left + 1;
                res[1] = right + 1;
                return res;
            }
        }
        return res;
    }
}```
