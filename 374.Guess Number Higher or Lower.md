# Guess Number Higher or Lower
## 方法 binary search
## 复杂度
Time: O(log(n))
### 代码

```java
public class Solution extends GuessGame {
    public int guessNumber(int n) {
        if (n == 0) {
            return 0;
        }
        int left = 1, right = n;
        while (left <= right) {
            int mid = left + (right - left)/2;
            if (guess(mid) == 0) {
                return mid;
            } else if (guess(mid)  == 1) {
                left = mid + 1;

            } else {
                right = mid -1;
            }
        }
        return -1;
    }
}
```
